## Slot Machine 🎰

A simple JS Slot Machine. Open HTML-File in your browser of choice, roll the slots, and become a virtual millionaire!!

Initial credit: 50$  
Fee per Roll: 1$  
Gain per Win: 20$ 🎉

## Features

- Casual gaming
- Responsive design

## Tech stack

- Javascript
- HTML5
- CSS3

To start rolling, clone repository and run "npm run dev".
